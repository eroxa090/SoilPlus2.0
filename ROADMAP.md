# SoilPlus 2.0 — Roadmap

> IoT + AI precision-agriculture platform for Central Asia.
> Sultan Bekkaliev & Ersultan Muratkali · Уральск, Казахстан · 2026

---

## Phase 0 — Текущее состояние (готово ✅)

- Веб-приложение на Next.js 15 + React 19, TailwindCSS, Supabase.
- Аутентификация: PBKDF2 + HMAC-cookie, первый пользователь = admin.
- AI-агроном (Gemini 2.5 Flash): чат, диагностика по фото, прогноз урожая.
- Расчёт полива по формуле KazNIIZ + FAO-56 (с расписанием по 35 л/м² за приём).
- Магазин устройств с заказами в Supabase.
- Страница /about с описанием проекта (без академических маркеров).
- Полный аудит безопасности (SECURITY.md).
- 11 культур в базе (пшеница, ячмень, кукуруза, томат, картофель, апорт, саксаул, тюльпан и др.).

## Phase 3 — Продуктовое расширение

**Цель:** превратить SoilPlus из MVP в платформу.

### AI / агрономия
- [ ] Расширить базу культур: до 30+ (бахчевые, виноград, лук, морковь, баштан).
- [ ] История измерений: графики pH/влажности/EC за 7/30/90 дней.
- [ ] Прогноз погоды от Open-Meteo → корректировка плана полива.
- [ ] Уведомления в Telegram-бот: "пора поливать", "EC растёт — промой".
- [ ] Распознавание стадии роста по фото (Gemini Vision + few-shot).

### Железо
- [ ] Финализировать прошивку ESP32 (текущая в research-папке).
- [ ] Серийная сборка партии 50–100 устройств.
- [ ] LoRa-шлюз для полей без Wi-Fi.
- [ ] Power: солнечная панель + LiFePO4, автономия ≥30 дней.
- [ ] OTA-обновления прошивки через Supabase Storage.

### Платформа
- [ ] Multi-tenant: один аккаунт = несколько участков, у каждого свои датчики и культуры.
- [ ] Роли: владелец / агроном / рабочий.
- [ ] Экспорт отчётов в PDF/Excel (для субсидий и страхования).

## Phase 4 — Масштаб

**Цель:** выйти за пределы Казахстана.

- [ ] Мобильное приложение (React Native или PWA-обёртка).
- [ ] Локализация: казахский, узбекский, английский.
- [ ] Интеграция с Минсельхоз РК (учёт субсидий за капельный полив).
- [ ] B2B-договоры с агрохолдингами: API-доступ + white-label.
- [ ] Партнёрство с производителями удобрений: рекомендации привязаны к их линейке.
- [ ] Маркетплейс: фермеры покупают/продают семена и продукцию через платформу.
